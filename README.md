## 🚀 Getting Started

### Prerequisites
* [Node.js](https://nodejs.org/) installed on your machine.
* [npm](https://www.npmjs.com/) (usually comes with Node.js).

### Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nishilchavda/Airstro.git
   cd Airstro
   ```

2. **Start the Backend:**
   ```bash
   cd b
   node server.js
   ```

3. **Start the Frontend (in a new terminal window):**
   ```bash
   cd f
   npm run dev
   ```
